2025.07.14

領域驅動設計與UX設計師的角色貢獻

講師：陳靜儀 / 3 小時

以「領域驅動設計（Domain-Driven Design, DDD）」與 UX 設計師在生成式 AI 發展中的角色為核心，透過講座與概念導入，引導參與者重新思考 AI 與使用者經驗設計之間的關係。探討生成式 AI 在當代數位產品中的應用前景，並解析 UX 設計師如何透過「語意建模」、「使用者意圖理解」與「交互場景設計」等方式，介入 AI 系統的建構與優化過程。

工作坊 設計603 11人參與
2025.07.15

LLM 的下一步？從 Agent 聊聊 Agentic AI

講師：張維元 / 3 小時

聚焦於大型語言模型（LLM）技術的最新發展趨勢，深入探討「Agent」與「Agentic AI」的概念與應用。介紹 LLM 技術的演進脈絡，涵蓋 RAG（Retrieval-Augmented Generation）、Fine-tuning、以及 AI Agent 的基本架構與運作方式。以資料分析場景為例，解析 Agent 工作流程的實作方式，並說明如何透過模組化的任務拆解、自主任務規劃與執行，實現具備決策能力的 AI 應用模型。

工作坊 設計106 13人參與
2025.07.22 上午

Agent 實作工作坊

講師：張維元 / 3 小時

延續前一場講座的實作工作坊，聚焦於 AI Agent 的開發與應用技術。從開發環境的建立開始，帶領學員操作 Vibe Coding 平台，進行環境安裝與設定。深入講解 Function Calling 機制，說明如何透過 LLM 與 API 的整合，使 Agent 能夠根據使用者意圖，自動完成特定任務。進一步延伸至 Multi-Agent 系統的設計，介紹 MCP（Model Context Protocol）協定，並說明多個 Agent 如何協同合作、分工執行不同任務。

工作坊 設計603 13人參與
2025.07.22 下午

領域驅動設計與UX設計師的角色貢獻工作坊 II

講師：陳靜儀 / 3 小時

介紹「使用案例（Use Case）」、「使用者流程（User Flow）」與「領域故事敘述（Domain Storytelling）」等方法，並透過簡報與 Miro 白板協作，實際演練如何在多方協作中進行需求與流程的視覺化。學員分組進行領域故事敘述的實作，模擬真實專案情境，協同定義業務流程與使用者互動場景。強調實作與協作，讓學員在實際操作中體驗領域驅動設計的應用。

工作坊 設計603 11人參與
2025.08.17

Google AI 生態系

講師：劉育維 / 6 小時

深入介紹 Google 的 AI 技術與生態系，實作多項無需撰寫程式碼的自動化工作流程。介紹 Google 的 AI Agent 概念，示範如何建立 Agentic AI Workflow，實現自動化任務處理。學習 Google Stitch，透過簡單的文字提示或圖像輸入，快速生成複雜的 UI 設計與前端程式碼，並將設計匯出至 Figma 進行進一步編輯。涵蓋 Google Firebase Studio 的應用，學習如何建立全端 AI 應用程式。

工作坊 設計603 12人參與
2025.09.02

生成式AI 產品開發社群成果發表

主持人：陳靜儀 / 3 小時

學員實作成果的展示與交流，作為整個系列工作坊的總結與驗收。學員將過去五場工作坊中所學的知識與技能，特別是生成式 AI、Agent 系統、領域驅動設計與 Google AI 生態系工具，應用於個人化平台的開發中。每位學員皆使用 Firebase Studio 建置自己的產品原型，完整呈現工作流程與功能。透過互動討論與意見交流，檢視彼此的學習成果，吸取不同的設計思路與技術應用方式。

本社群成果

除了順利誕生 10 個小型 AI 產品原型，展現跨領域學員在短時間內的高速學習能力外，更有 2 位學員成功通過 iPAS「AI 應用規劃師（初級）」檢定（共有七位報考），以實力證明了持續精進的專業態度。

同時，社群能量也成功延伸到國際舞台。教師陳靜儀將工作坊的重要成果整理為研究論文，成功投稿至 2026 AEIT（Advances in Education and Information Technology）國際研討會，並將於 2026 年 1 月 23–25 日於日本札幌進行口頭發表。

論文題為：Teaching UX Evaluation for AI Products: A Reflective Workshop Framework for Design Education（AI產品使用者體驗評估教學：設計教育中的反思式工作坊框架）。
